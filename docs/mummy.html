    mummy.js反其道而行，扩展「污染」原生的JS对象。 以实现更简洁，更直观的操作 没必要像主流JS框架那样，去封装一层，比如：

    var $el = $(dom);
var el = $el.get(0);

    封装后再取原生的dom对象就需要用指定的方法操作
    不管是书写上还是从初学者理解上，显得都比较繁琐

    mummy.js要借鉴了以下两个很优秀的库的思想
